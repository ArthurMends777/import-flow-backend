# Import SaaS Backend

Sistema para automação de processos de importação (DI → NF-e).

## 🚀 Tecnologias
- Node.js
- NestJS
- PostgreSQL
- Prisma

## 📦 Funcionalidades
- Upload de DI
- Processamento de dados
- Geração de NF-e (XML)

## 🔐 Segurança
- Autenticação JWT
- Hash de senha com bcrypt
- Multi-tenant
